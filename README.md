# EJS Backend laboration 1

##  [Render EJS Lab 1](https://backendejs-ucl7.onrender.com/)

## git clone https://github.com/isasod2500/EJSBackendlab1

Backend, serverbaserat kurs CV.
HTML är skapat med hjälp av .ejs och styrs av js kod från main.js. Rendering av undersidor möjliggörs av get och render vid knapptryck eller länktryck. 
Formuläret på undersidan "Lägg till kurs" är av POST format och kontrollerar med SQL-databasen att inlägg inte dupliceras, så som kurskod. Länkar jämförs med kurskoden och nekas i fall dem inte stämmer överens. 
Vid borttagning av kurser från CVt finns en knapp tillhörande varje kurs som tar bort inlägget från webbplatsen samt databasen.
